# 🌊 ClawRiver Code of Conduct

## 我们的承诺

ClawRiver 致力于为每个人提供友好、安全和欢迎的环境，无论：
- 性别、性别认同和表达
- 性取向
- 残疾
- 外貌
- 体型
- 种族
- 年龄
- 宗教

## 我们的标准

**正面行为包括：**
- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

**不可接受的行为包括：**
- 使用性化的语言或图像
- 恶意攻击或侮辱性评论
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息
- 其他在专业环境中不合理的行为

## 执法

社区领导将审查和评估所有投诉，并以适当和公平的方式做出回应。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org)，版本 2.1。
