### 关键算法

**1. Cross-Encoder 预测**:
```python
# 输入: [query, doc1], [query, doc2], ...
pairs = [[query, text1], [query, text2], ...]

# 批量推理
scores = encoder.predict(pairs)  # [0.92, 0.45, 0.23, ...]

# 阈值过滤
filtered = [(doc, score) for doc, score in zip(docs, scores) if score >= threshold]

# Top-K 排序
top_k = sorted(filtered, key=lambda x: x[1], reverse=True)[:20]
```

**2. MRR 计算**:
```python
def calculate_mrr(ranked_ids, ground_truth):
    for i, rid in enumerate(ranked_ids):
        if rid in ground_truth:
            return 1.0 / (i + 1)
    return 0.0
```

**3. NDCG 计算**:
```python
def calculate_ndcg(ranked_ids, ground_truth, k=10):
    # DCG
    dcg = sum(1.0 / log2(i + 2) for i, rid in enumerate(ranked_ids[:k])
              if rid in ground_truth)

    # IDCG
    idcg = sum(1.0 / log2(i + 2) for i in range(min(k, len(ground_truth))))

    return dcg / idcg if idcg > 0 else 0.0
```

## 风险和挑战

### 已识别风险

1. **模型下载失败**
   - **原因**: Hugging Face 网络问题
   - **缓解**: 使用镜像站点、预打包模型

2. **内存占用**
   - **原因**: 模型大小 ~1.3GB
   - **缓解**: 模型量化、减少 batch size

3. **延迟波动**
   - **原因**: 缓存未命中时延迟增加
   - **缓解**: 预热缓存、异步重排

4. **相关性未提升**
   - **原因**: 候选质量差、阈值过高
   - **缓解**: 优化向量搜索、调整阈值
