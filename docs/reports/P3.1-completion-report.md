**个性化得分计算**:
```python
personalization_score = (
    language_match * 0.1 +
    interest_match * 0.2 +
    tech_stack_match * 0.1 +
    editor_preference * 0.15 +
    theme_preference * 0.1
)

final_score = base_relevance * 0.8 + personalization_score * 0.2
```

## 技术实现细节

### 两层画像架构

```
静态事实层 (UserProfile)
├── 个人信息
├── 偏好
├── 习惯
├── 技能
└── 兴趣

动态上下文层 (UserDynamicContext)
├── 当前状态
├── 工作状态
├── 最近活动
└── 推荐信息
```

### 自动提取流程

```
对话输入
    ↓
LLM/规则提取
    ↓
置信度过滤 (≥0.6)
    ↓
与现有事实对比
    ↓
识别新事实/更新
    ↓
写入数据库 + 缓存失效
    ↓
记录变更历史
```

### 性能优化策略

1. **Redis 缓存**:
   - 画像查询: <50ms
   - 动态上下文: <50ms
   - TTL: 300秒

2. **数据库优化**:
   - 索引设计: 8个复合索引
   - 预加载: selectinload
   - 批量查询: 减少往返

3. **异步处理**:
   - 所有数据库操作异步
   - 非阻塞 LLM 调用
   - 并发提取优化
