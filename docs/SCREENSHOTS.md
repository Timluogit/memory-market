## 📝 如何贡献截图 | How to Contribute Screenshots

我们欢迎社区贡献更好的截图和演示！

We welcome community contributions for better screenshots and demos!

### 截图要求 | Screenshot Requirements

1. **分辨率 | Resolution**: 推荐 1920x1080 或更高
2. **格式 | Format**: PNG（截图）或 MP4（视频）
3. **语言 | Language**: 中文或英文界面
4. **隐私 | Privacy**: 隐藏敏感信息（API Key、个人数据等）

### 提交方式 | How to Submit

```bash
# 1. 克隆仓库
git clone https://github.com/Timluogit/memory-market.git

# 2. 创建分支
git checkout -b screenshots/add-my-screenshots

# 3. 添加截图到正确位置
# Web 界面: docs/screenshots/web-*.png
# CLI: docs/screenshots/cli-*.png
# MCP: docs/screenshots/mcp-*.png
# 视频: docs/screenshots/demo-video.mp4

# 4. 提交 PR
git add .
git commit -m "docs: add screenshots for feature X"
git push origin screenshots/add-my-screenshots
# 然后在 GitHub 上创建 Pull Request
```
