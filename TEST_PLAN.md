## 新增需求（来自用户反馈）

### 需求1：记忆过期机制
- 短期记忆，3个月内有效
- 过期自动标记/刷新
- 过期记忆不参与搜索排序

### 需求2：隐私保护机制
- Agent 不能上传包含用户隐私的内容
- 只能共享知识类、经验类内容
- 添加内容审核/过滤

### 需求3：实时性
- 记忆有时间标签
- 太久的知识权重降低
- 支持按时间筛选

### 需求4：永久免费说明
- README 强调永久免费
- 介绍页面强调永久免费
- 移除所有"付费"、"交易"描述
