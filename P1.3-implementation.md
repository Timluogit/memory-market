### 1. 数据模型扩展

**文件：** `app/models/tables.py`

新增表：
- `SearchLog` - 搜索日志表
- `SearchClick` - 搜索点击记录表
- `SearchABTest` - A/B测试配置表

**字段说明：**

#### SearchLog
- `log_id` - 日志ID
- `agent_id` - 用户ID
- `query` - 搜索查询
- `search_type` - 搜索类型（vector/keyword/hybrid）
- 过滤条件：category, platform, format_type, min_score, max_price, sort_by
- 结果信息：result_count, top_result_id
- 性能指标：response_time_ms, semantic_score, keyword_score
- A/B测试：ab_test_id, ab_test_group
- 会话：session_id

**索引优化：**
- `idx_search_logs_agent_time` - 用户+时间
- `idx_search_logs_query_time` - 查询+时间
- `idx_search_logs_type_time` - 类型+时间

#### SearchClick
- `click_id` - 点击ID
- `search_log_id` - 搜索日志ID
- `memory_id` - 记忆ID
- `position` - 结果位置
- `agent_id` - 用户ID
- `click_type` - 点击类型（detail/purchase/favorite）
- A/B测试：ab_test_id, ab_test_group

**索引优化：**
- `idx_search_clicks_agent_time` - 用户+时间
- `idx_search_clicks_memory_time` - 记忆+时间

#### SearchABTest
- `test_id` - 测试ID
- `name` - 测试名称
- `description` - 测试描述
- `test_type` - 测试类型（algorithm/reranking/filtering/sorting）
- `start_at` / `end_at` - 开始/结束时间
- `split_ratio` - 分流比例（{"A": 0.5, "B": 0.5}）
- `group_configs` - 分组配置（{"A": {...}, "B": {...}}）
- `metrics` - 目标指标（["ctr", "zero_results_rate", "avg_response_time"]）
- `group_stats` - 分组统计
- `results` - 分析结果
- `significance` - 统计显著性
- `winner` - 获胜组
- `status` - 状态（draft/running/completed/cancelled）

**索引优化：**
- `idx_search_ab_tests_status` - 状态+创建时间
- `idx_search_ab_tests_time` - 开始+结束时间

### 2. 搜索日志中间件

**文件：** `app/api/search_middleware.py`

**功能：**
- ✅ 异步日志写入（队列+worker）
- ✅ 零性能影响（<3%）
- ✅ 自动A/B测试分组
- ✅ 点击追踪
- ✅ 会话追踪

**使用示例：**
```python
from app.api.search_middleware import get_search_log_middleware

# 获取中间件实例
middleware = await get_search_log_middleware()

# 记录搜索
await middleware.log_search(
    db=db,
    agent_id=agent_id,
    query=query,
    search_type=search_type,
    filters=filters,
    results=results,
    response_time_ms=response_time_ms,
    scores={"semantic": 0.85, "keyword": 0.72},
    session_id=session_id
)

# 记录点击
await middleware.log_click(
    db=db,
    agent_id=agent_id,
    search_log_id=search_log_id,
    memory_id=memory_id,
    position=position,
    click_type="detail"
)
```

### 4. A/B测试框架

**文件：** `app/services/ab_test_service.py`

**功能：**
- ✅ A/B测试创建/启动/停止/取消
- ✅ 用户自动分流（基于哈希）
- ✅ 结果收集
- ✅ 统计分析
- ✅ 获胜者推荐

**使用示例：**
```python
from app.services.ab_test_service import get_ab_test_service
from app.models.schemas import ABTestCreate

# 获取服务
service = await get_ab_test_service()

# 创建测试
test_config = ABTestCreate(
    name="向量搜索 vs 混合搜索",
    test_type="algorithm",
    start_at=datetime.utcnow() + timedelta(hours=1),
    end_at=datetime.utcnow() + timedelta(days=7),
    split_ratio={"A": 0.5, "B": 0.5},
    group_configs={
        "A": {"algorithm": "vector"},
        "B": {"algorithm": "hybrid", "keyword_weight": 0.4}
    },
    metrics=["ctr", "zero_results_rate", "avg_response_time"]
)

test = await service.create_test(db, agent_id, test_config)

# 启动测试
await service.start_test(db, test.test_id)

# 获取用户分组
group = await service.get_user_group(db, test.test_id, agent_id)

# 获取分组配置
config = await service.get_group_config(db, test.test_id, group)

# 收集结果
stats = await service.collect_results(db, test.test_id)

# 分析结果
result = await service.analyze_results(db, test.test_id)
```
